# TeleCaller - AI Voice & WhatsApp Automation SaaS Platform

Multi-tenant SaaS platform for AI-powered voice calling and WhatsApp automation.

## Features

- AI Voice Calling with customizable agents
- WhatsApp Business API integration  
- Campaign management and lead tracking
- Call analytics and compliance (DNC lists)
- Credit-based billing system

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Express, TypeScript
- **Database**: PostgreSQL with Drizzle ORM
- **Integrations**: SignalWire, Deepgram, ElevenLabs
